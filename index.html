Documentação - Torne Seu Terminal Linux Incrível

Introdução

Enquanto o prompt padrão no terminal Linux é um pouco chato, existem opções para personalizá-lo facilmente. Felizmente, uma quantidade razoável de personalização pode ser feita a partir do terminal para torná-lo visualmente atraente e mais utilizável.

Este documento irá guiá-lo sobre como fazer seu terminal Linux parecer fabuloso usando o Fish shell com OhMyPosh.

Fish Shell

O Fish shell é um shell amigável com sugestões automáticas, destaque de sintaxe e uma linguagem de script simples. É poderoso, intuitivo e ideal para iniciantes e usuários avançados.

OhMyPosh

OhMyPosh é um mecanismo de tema de prompt universal para qualquer shell com glória de pesca. Ele fornece uma infinidade de prompts personalizáveis que exibem informações úteis como status do git, informações do sistema e muito mais.

Instalação do Fish Shell

Ubuntu/Debian/Linux Mint

sudo apt install fish -y

Fedora

sudo dnf install fish -y

Arch Linux / Manjaro Linux

chsh -s /usr/bin/fish

Executando o Fish Shell

Instalando Oh My Posh

sudo wget https://github.com/JanDeDobbeleer/oh-my-posh/releases/latest/download/posh-linux-amd64 -O /usr/local/bin/oh-my-posh
sudo chmod +x /usr/local/bin/oh-my-posh

Baixando e Instalando Fontes

mkdir -p $HOME/.local/share/fonts
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/FiraCode.zip -O $HOME/Downloads/firacode.zip
unzip $HOME/Downloads/firacode.zip -d $HOME/.local/share/fonts
fc-cache -f -v

Adicionando Tema OhMyPosh ao Config do Fish

gnome-text-editor $HOME/.config/fish/config.fish
oh-my-posh init fish --config $HOME/.poshthemes/montys.omp.json | source
